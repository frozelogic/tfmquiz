# Quiz de Perguntas do Transformice

Código Lua para sistema de Quiz de Perguntas no Transformice. Originalmente gerenciado por Shun_kazami#7014, porém com sua saída o código ficou abandonado. Após alguns dias, tomei a iniciativa de cuidar e adaptar o sistema para mantê-lo ativo e funcional.

## Sobre o Projeto

Este script oferece um sistema dinâmico de perguntas e respostas para diversão dentro do Transformice. Qualquer jogador pode utilizá-lo livremente!

## Principais Atualizações

• **Novas perguntas:** Expansão contínua do banco de questões.\
• **Revitalização de perguntas:** Revisão e melhoria do conteúdo existente.\
• **Novas funções:** Aprimoramento da experiência do usuário e jogabilidade.

## Como Usar

O código é livre para uso de jogadores normais e FunCorpers. Basta acessar o arquivo, copiar todo o seu conteúdo, e colar no **/lua**. Lembre-se de colocar o nome dos administradores da sala no **admin = {""}**.

## Contribuições
